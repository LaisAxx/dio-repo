# Dio Bank

Projeto Dio Bank é uma aplicação de transferência bancárias com .NET, para treinar conceitos de orientação a objetos na prática.

### Features

- [x] Listar Contas
- [x] Inserir nova conta
- [x] Tranferir
- [x] Sacar
- [x] Depositar
- [x] Limpar Tela
- [x] Sair da aplicação
